# ***editor.rulers***

```json
{
    "editor.rulers": [
      80
    ]
}
```

**Descripción:** *Coloca una regla vertical en la columna especificada. Esto ayuda a mantener el ancho del código dentro de un límite recomendado (por ejemplo, 80 caracteres).*
