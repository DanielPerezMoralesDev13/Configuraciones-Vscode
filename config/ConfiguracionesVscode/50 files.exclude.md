# ***files.exclude***

```json
{
  "files.exclude": {
    "**/.git": false,
    "**/.svn": false,
    "**/.hg": false,
    "**/CVS": false,
    "**/.DS_Store": false,
    "**/node_modules": false
  }
}
```

**Descripción:** *Excluye ciertos directorios y archivos del árbol de archivos del explorador, como `.git`, `.svn`, `.hg`, entre otros.*
